# Supply Chain Analytics Project

## Overview
This project demonstrates KPI analysis for a supply chain sales dataset. 
It includes revenue, cost, and profit analysis using Python and SQL.

## Tools Used
- Python (Pandas)
- SQL
- KPI Framework Design
- Business Performance Metrics

## Files Included
- sales_data.csv : Sample dataset
- analysis.py : Python script for KPI calculations
- kpi_query.sql : SQL query example for regional KPIs
- requirements.txt : Python dependencies

## KPIs Calculated
- Total Revenue
- Total Profit
- Average Margin
- Regional Performance Summary

## How to Run
1. Install dependencies:
   pip install -r requirements.txt

2. Run analysis:
   python analysis.py

## Purpose
This repository demonstrates core data analytics skills including:
- Data cleaning and transformation
- KPI development
- Financial metric evaluation
- SQL aggregation logic
